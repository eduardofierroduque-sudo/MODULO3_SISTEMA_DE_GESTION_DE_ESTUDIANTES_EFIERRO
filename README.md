Este es un simulador básico desarrollado en Python para practicar la manipulación de estructuras de datos. Permite gestionar nombres de estudiantes, sus calificaciones, y las asignaturas disponibles.
Describe los pasos que realiza tu script:
Gestión de Listas: Se añade un estudiante nuevo (.append) y se modifica uno existente mediante su índice.
Mapeo de Datos: Se asocian estudiantes con sus notas usando un diccionario para búsquedas rápidas.
Lógica de Conjuntos: Se gestionan materias obligatorias y optativas, utilizando la operación de Unión para ver el catálogo completo.
Validaciones: El sistema verifica si un alumno o materia específica existe antes de operar.
